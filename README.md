# Homebrew Tap

Homebrew formulae for Voxly CLI tools.

## Install

```bash
brew tap outerbanks73/tap
brew install getscript
```

## Tools

| Formula | Description |
|---------|-------------|
| [getscript](Formula/getscript.rb) | Fast, Unix-friendly CLI for fetching transcripts from YouTube and Apple Podcasts |

## Updating

```bash
brew update
brew upgrade getscript
```
